# IoT Device Management System - User Guide

## Overview
This IoT Assessment application is a comprehensive device management system built with ASP.NET Core 8.0, featuring real-time device monitoring, MQTT communication, and modern Tailwind CSS interface.

## 1. How to Use the System

### Getting Started
1. **Access the Application**: Navigate to `http://localhost:5212` in your web browser
2. **Dashboard Overview**: The main dashboard provides:
   - Real-time device statistics (Total, Online, Offline, Low Battery)
   - Recent device activity and status updates
   - Device distribution by type and location
   - Low battery alerts with visual indicators
   - Quick action buttons for device management

### Navigation
- **Dashboard**: Overview of all device metrics and recent activity
- **Devices**: Manage your IoT devices (View, Add, Edit, Delete)
- **Logs**: View system logs and device activity history
- **Footer Links**: Quick access to device management and logs

### Key Features
- **Real-time Updates**: Dashboard auto-refreshes every 30 seconds
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Modern UI**: Beautiful Tailwind CSS interface with animations and hover effects
- **MQTT Integration**: Real-time device communication via standardized topics
